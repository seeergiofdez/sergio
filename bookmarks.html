Bookmarks

An unattended list of some tools and online pages that I find extremely useful.

  • dogbin. To take notes and share it with other people
  • photopea. Edits images online
  • katbin. A fork of deletescape's dogbin
  • filegarden. Hosts and upload images and other files, useful for this site
  • squoosh. Compresses PNG, JEPG and WEBP files, works offline
  • beatufier. Beautifies JS and CSS code
  • alphabetizer. Alphabetizes, order and other tools
  • html validator. Validates HTML code
  • dns lookup. Lookups DNS records from any site
  • diffchecker. Git-style commit diff checker but for any text
  • contrast checker. Checks whether color contrast passes AA and AAA
  • squiwtf. Downloads music up to 24-bit 192kHz from tidal
  • binilossless. An alternative to download lossless music
  • monochrome. Another alternative to download lossless music
  • qobuz squidwtf. Upper link but downloads from qobuz

List updated on: January 9, 2025
