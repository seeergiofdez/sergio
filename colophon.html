Colophon

There is nothing really interesting in here. This is just the updated version of Sergio's Personal Cabinet, as I want to separate the things I do on the BBJProjeK organization, and my own thoughts. Cabinet will still be updated for The Authority docs, and XasPos is still going to update there.

Here you'll find some personal thoughts and other stuff.

Website

As on previous generations, the website is built from scratch, all the HTML and CSS is manually written by me (although I grabbed some css ideas from other blogs), and the whole repository is publicly available at seeergiofdez/sergio↗.

In my desire to make a really lightweight but modern html5 website (with pages around 8-36 kB without images), I had to divide stylesheets and scripts so this way I only add the necessary features for each HTML page. This was already a thing on cabinet, and still following this principle.

Cookies and tracking

No, the page does not store cookies. If you find one, it’s probably a bug, and I’d appreciate if you’d let me know. No external info or iframes will be shown here.

I don’t use any kind of analytics. This page is hosted on GitLab Pages and I try not to hotlink to external sites. Also, as I always did, this page is designed to last. I use almost every principle that Jeff Huang specifies in their post: This Page is Designed to Last↗. I use vanilla HTML/CSS, I use as few pages as possible to retain information, I don't hotlink, and I use native fonts.

Why under bbjprojek.org domain

It's an excellent question that I don't know how to address, I am too lazy to find a personal domain that convinces me, and for me, bbjprojek was initially a personal project, and so I will always use this domain as a personal thing, as well as my email.
